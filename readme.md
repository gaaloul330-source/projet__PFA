# 🚀 Projet de Fin d’Année : Automatisation des Ventes

## 📝 Présentation du Projet
Ce projet a été réalisé dans le cadre du cours *Logiciels* à la *Faculté des Sciences de Tunis* [1, 2]. 

L'objectif est d'aider une entreprise de e-commerce à automatiser son suivi des ventes en migrant d'Excel vers *Python* pour gérer plus efficacement de gros volumes de données [1].

## 🛠️ Fonctionnalités du Script
Le script Python développé permet d'automatiser l'ensemble du flux de traitement des données :

1.  *Génération automatique* 📁 : Création du fichier ventes.csv (ID, Prix, Quantité, Remise) [3].
2.  *Calculs de Performance* 📊 :
    *   *CA Brut* : Calcul automatique (Prix × Quantité) [3].
    *   *CA Net* : Application des remises en pourcentage [3].
    *   *TVA* : Calcul automatique de la taxe (20%) [3].
3.  *Analyse des Résultats* 📈 :
    *   Affichage du *Chiffre d’Affaires Total* de l’entreprise [3].
    *   Identification du produit générant le *plus gros bénéfice* [3].
4.  *Exportation* 📤 : Génération d'un rapport final nommé resultats_final.csv incluant tous les nouveaux indicateurs calculés [2, 3].
